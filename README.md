# UE-GameFeaturePractice

Unreal Engine 5.4 환경에서 게임 기능 구현과 설계 역량을 향상시키기 위한 개인 연습 레포입니다.  
본 레포는 티켓 기반 기능 설계 → 구현 → 검증 흐름을 반복하며,  
실제 개발자가 기능을 분석하고 구조를 설계하는 사고 과정을 훈련하는 것을 목표로 합니다.

특히 이 레포는 AI를 단순한 코드 생성 도구가 아닌,  
설계 토론 파트너이자 검증 도구로 활용하는 협업 연습을 함께 포함합니다.

훈련 방식은 다음과 같습니다.

1. AI로부터 기능 구현 티켓을 발행받음  
2. 티켓을 기반으로 직접 설계를 작성  
3. 설계 과정에서 발생할 수 있는 구조적 문제를 AI와 토론하며 보완  
4. 최종 설계를 기준으로 AI로부터 구현 코드를 제공받아 Unreal Engine에서 테스트  
5. 구현 결과와 문제점을 정리하며 설계와 구현 역량을 동시에 개선

이를 통해  
- 기능 단위 설계 능력  
- Unreal Engine 구조 이해  
- 문제 분석 및 수정 능력  
- AI와의 협업 기반 개발 프로세스  
를 함께 훈련하는 것을 목표로 합니다.

이 레포는 완성된 결과물보다,  
설계 사고 과정과 문제 해결 흐름 자체를 기록하는 학습용 프로젝트입니다.

## How to Use
1) `Progress_Log.md`를 최신으로 유지  
2) 매일 티켓 발행 후 `Docs/Days/DayXXX_...`에 기록  
3) 구현/검증은 `Sandbox/` Unreal 프로젝트에서 실행

## Folder Guide
- `Docs/Templates/` : 문서 템플릿
- `Docs/Days/` : 일자별 티켓 기록
- `Sandbox/` : Unreal 테스트 프로젝트
- `Progress_Log.md` : 완료/진행 인덱스

## Conventions
- Day 폴더명: `Day###_<CATEGORY>-<ID>`
- 티켓 ID 예: `SKILL-001`, `NET-003`
